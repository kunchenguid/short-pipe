# Assets

Short Pipe is a **wordmark-only** brand. The product repo ships **no logo image, icon set,
illustration, or photography** — so there are no raster/vector brand files to copy in here.

What stands in for "assets":

- **Logo** — the wordmark **"Short Pipe"** typeset in *DM Serif Display* (regular), optionally
  with the tagline *long-form in, captioned shorts out* in Inter. See `preview/brand-wordmark.html`
  for the light and on-dark lockups. To use it, just set `font-family: var(--font-display)`.
- **Icons** — **Lucide** line icons (real path data, inlined in `ui_kits/short-pipe/data.jsx`).
  This is a documented substitution; the repo has no icon set. See README §4 (Iconography).
- **Imagery** — there is none by design. The only "imagery" is the user's own video frames,
  shown full-bleed (cropped) or inside the paper card. In mocks, use a neutral charcoal frame
  (`--frame`), never a colored placeholder.

If the team later adds a real logo or icon set, drop the files here and update README §4.
